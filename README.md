# Shadeseeker
## Design and Code for Shadeseeker Robot

### Purpose:
   The Robot is designed to roam around and find the darkest spots in a room.
    
### Materials:
- Arduino Uno Board
- Breadboard
- DC Gearbox Motors (2x)
- Ultra Sonic Sensors
- light sensors (3x)
- RGB LED
- Caster (for third wheel)
- IR sensor / remote
- PowerBank Cell Phone Charger
- H Bridge
- Resistors
- Wheels (2x)
